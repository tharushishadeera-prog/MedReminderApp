MedReminder App 💊
This is a mobile application developed to help users manage their medicine schedules effectively. It provides timely reminders to ensure users never miss a dose.

Features
User Authentication: Secure sign-up and login using Firebase Authentication.

Medicine Management: Easily add, view, and manage your daily medications.

Smart Reminders: Local notifications to alert users when it's time to take their medicine.

History Tracking: Keep track of your medication history.

Technology Stack
Framework: Expo (React Native)

Backend: Firebase (Firestore & Authentication)

Notifications: Expo Notifications

Deployment: EAS Build

Setup Instructions
Clone the repository:

Bash
git clone <your-repo-link>
cd MedReminderApp
Install dependencies:

Bash
npm install
Configure Firebase:

Create a firebaseConfig.ts file inside the firebase/ directory.

Add your Firebase project credentials.

Run the app:

Bash
npx expo start
How to Test
Using Expo Go: Scan the QR code generated in your terminal (Note: Push notifications require a Development Build).
